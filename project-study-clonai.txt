в—Ҹ Here's a complete breakdown of the authentication system:

  ---
  How Login Works

  Flow

  User fills sign-in form
    вҶ’ authClient.signIn.email() (or Google OAuth)
    вҶ’ Next.js route handler at /api/auth/[...all]
    вҶ’ Better Auth processes credentials
    вҶ’ Session created in Convex `session` table
    вҶ’ ensureActiveOrganization() called
    вҶ’ Redirected to /dashboard
    вҶ’ OrganizationGuard checks activeOrganizationId
    вҶ’ If no active org вҶ’ OrgSelectionView
    вҶ’ If active org вҶ’ Dashboard shown

  Entry Points

  - Sign-in page: apps/web/modules/auth/ui/views/sign-in-view.tsx вҖ”
  email/password form + Google OAuth button
  - Sign-up page: apps/web/modules/auth/ui/views/sign-up-view.tsx вҖ” same
  options, creates account
  - Org selection: apps/web/modules/auth/ui/views/org-selection-view.tsx вҖ” shown
   when a user belongs to multiple organizations
